# RaulTCC

Academic final project (TCC) developed for [FATEC RP](http://www.fatecrp.edu.br/)).

## About

This project is a **search and automation tool** built with JavaScript (Node.js) and AutoHotkey. It was developed as part of the requirements for the Technology graduation course at FATEC.

The AutoHotkey component (`Search_Tool.ahk`) handles desktop automation, while the Node.js backend (`index.js`) supports the search logic and processing.

## Tech Stack

- **JavaScript / Node.js** — backend logic and search processing
- **AutoHotkey** — desktop automation scripting - V1.1

## Getting Started

```bash
# Install dependencies
npm install

# Run the application
node index.js
```

For the AutoHotkey component, make sure [AutoHotkey](https://www.autohotkey.com/) is installed, then run `Search_Tool.ahk` directly.
